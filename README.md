## Quick  Start
1. `$ npm i`
2. `$ npm start`
3. Access [http://localhost:8080/](http://localhost:8080/)

## Command
Action | Command
--|--
Start Dev | `$ npm start`
Unit Test | `$ npm test:unit`
E2E Test | `$ npm test:e2e`
Build | `$ npm runn Build`
